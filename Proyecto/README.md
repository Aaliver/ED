# Proyecto: Codificación de Huffman (Variante)

# Objetivo

El objetivo del Proyecto ha sido crear una variante de la codificación Huffman que reduzca el tamaño de los datos de una cadena de caracteres con el arbol Huffman correspondiente al codificar y decodificar de bits a cadena la respectiva cadena.

# Funcionamiento



# Ejecución

{-
let arbolito = arbolito "AVRAHKADABRA"
let bits = encoding "AVRAHKADABRA" 
decoding bits arbolito
-}
